Инструкция для запуска/разворачивания сервера.

1) Проект скачивается из github по адресу https://github.com/agusev04/backgammon
2) Сервер находится в папке backgammon-server, перейти туда
3) Собрать проект - gradlew build
4) В папке build/distributions соберутся архивы, скопировать содержимое одного из архивов
5) В содержимом запустить скрипт bin/backgammon-server. Сервер поднимется на порту 8888.
